class cloudknot.aws.SecurityGroup(security_group_id=None, name=None, vpc=None, description=None)[source]¶

Class for defining an AWS Security Group

Initialize an AWS Security Group.

Parameters:
  • security_group_id (string) – ID of the security group to be retrieved
  • name (string) – Name of the security group to be created
  • vpc (Vpc) – Amazon virtual private cloud in which to establish this security group
  • description (string) – description of this security group if description == None (default), then description is set to “This security group was generated by cloudknot” Default: None

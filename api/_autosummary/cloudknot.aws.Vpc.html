cloudknot.aws.Vpc¶

class cloudknot.aws.Vpc(vpc_id=None, name=None, use_default_vpc=False, ipv4_cidr=None, instance_tenancy=None)[source]¶

Class for defining an Amazon Virtual Private Cloud (VPC)

Initialize a Vpc instance

Parameters:
  • vpc_id (string) – VPC-ID for the VPC to be retrieved
  • name (string) – Name of the VPC to be retrieved or created
  • use_default_vpc (bool) – if True, create or retrieve the default VPC if False, use other input args to create a non-default VPC
  • ipv4_cidr (string) – IPv4 CIDR block to be used for creation of a new VPC
  • instance_tenancy (string) – Instance tenancy for this VPC, one of [‘default’, ‘dedicated’] Default: ‘default’

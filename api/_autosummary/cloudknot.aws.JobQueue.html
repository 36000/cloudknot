class cloudknot.aws.JobQueue(arn=None, name=None, compute_environments=None, priority=None)[source]¶

Class for defining AWS Batch Job Queues

Initialize an AWS Batch job definition object.

Parameters:
  • arn (string) – Amazon Resource Number of the job definition
  • name (string) – Name of the job definition
  • compute_environments (ComputeEnvironment or tuple(ComputeEnvironments)) – ComputeEnvironment instance or sequence of ComputeEnvironment instances (in order of priority) for this job queue to use
  • priority (int) – priority for jobs in this queue Default: 1
